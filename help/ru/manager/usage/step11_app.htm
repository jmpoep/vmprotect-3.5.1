Шаг 1.1: Создаем защищаемое приложение

Первым шагом будет создание приложения. Это будет простое приложение, без интерфейса пользователя и каких-либо серьезных возможностей. Нашей задачей будет передать системе лицензирования серийный номер и получить от нее ответ.

#include <windows.h>
#include <stdio.h>

bool is_registered(const char *serial)
{
        return serial && serial[0] == 'X';
}

int main(int argc, char **argv)
{
        char *serial = "Xserialnumber"; // зададим номер непосредственно в коде для простоты
        if (!is_registered(serial))
        {
                printf("please register!\n");
                return 0;
        }
        printf("We are registered.\n");
        return 0;
}

Программа реализует очень простую схему проверки серийного номера. Функция is_registered() сравнивает первую букву номера с 'Х' и считает номер корректным в случае совпадения. Для некорректного номера выдается просьба о регистрации, а в случае правильного номера выводится сообщение "We are registered."
