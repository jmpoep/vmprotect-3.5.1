Система лицензирования

Система лицензирования решает задачи создания и проверки серийных номеров. Поддерживаются ограничения работы защищаемого ПО по дате и времени, привязка к оборудованию, шифрование кода серийным номером, ограничение периода бесплатных обновлений и многое другое. Система построена на асимметричных алгоритмах шифрования, что минимизирует вероятность создания генераторов серийных номеров. Для защиты системы лицензирования применяется виртуализация VMProtect, что минимизирует вероятность взлома на уровне кода и создания патчей.
