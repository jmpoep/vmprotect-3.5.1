Раздел "Проект"

Раздел "Проект" содержит следующие секции:
  • Функции для защиты - позволяет выбрать объекты для защиты
  • Лицензии - позволяет управлять лицензиями и серийными номерами
  • Файлы - позволяет включить в защищаемый EXE файл дополнительные файлы данных или DLL
  • Скрипт - позволяет расширить возможности защиты за счет использования встроенного скриптового языка на базе LUA
  • Опции - позволяет настроить параметры защиты приложения
